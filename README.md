# Utkarsh Portfolio

A personal portfolio built with Vite + React + TypeScript and Tailwind CSS.

## Project Structure

- `app/` - Main frontend app (Vite + React)

## Setup

```bash
cd app
npm install
```

## Run (Development)

```bash
npm run dev
```

## Build

```bash
npm run build
```

## Preview (Production Build)

```bash
npm run preview
```
